# DevClip UI (Angular)

This package is the **renderer** for DevClip. The canonical docs and scripts live in the **repository root** — see [`../README.md`](../README.md).

## Typical workflow

From the **repo root** (not only this folder):

```bash
npm install
cd angular-app && npm install && cd ..
npm run dev
```

Production UI build:

```bash
npm run build:ng
```

For Angular CLI details: [Angular documentation](https://angular.dev).
